# Tonesu

Tonesu is a constructed language built around one idea: **words should say what they mean**.

Every word in Tonesu is built from a small set of root syllables — around 34 of them — combined according to transparent rules. If you know the roots and the rules, you can read an unfamiliar word and get close to its meaning without looking it up. If you need a new concept, you build it rather than invent it.

The name says it all: **to-ne-su** — *pattern, relation, structure* — truth as organized knowledge. The language is named after what it aspires to express.

---

## A taste

```
la-mi  to  lo-tonesu
```
*I hold truth. / I know the truth.*

```
la-ze  lo-si  ka-sikipast  wi [ka-fesi  ne-yu]
```
*She sent the message to warn them.*

```
la-toli  seka  lo-tosu
```
*The scholar examines the theory.*
