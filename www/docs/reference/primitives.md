# Primitive Roots

These are the 34 foundational roots of Tonesu — the conceptual atoms from which all words are built. Every compound word is made from some combination of these.

Each root is a **CV syllable** (consonant + vowel). This shape is structurally distinct from longer compounds, so a root is always recognisable by its form.

---

## Entities and Substances

| Root | Gloss | Meaning |
|------|-------|---------|
| `mu` | object / artifact | physical non-living things: tools, devices, objects |
| `ma` | matter / substance | raw unformed material, physical substrate |
| `zo` | living thing | any organism: plant, animal, fungus, microbe |
| `li` | social agent / person | conscious intentional actor; implies `zo` unless combined with `mu` |
