# Stage 3 — Grammar and notation

Grammar particles and notation marks are taught together because learners
encounter them together. In real Tonesu, `go {premise}, result` involves a
particle *and* a structural bracket simultaneously — teaching them in separate
stages creates an artificial split.

Topics: `la-` / `lo-` / `lu-` in full; `ne` copula; `go` causal particle vs `;` sequential
connector; `/` parallel partition; `he` vocative; `ya` attention signal; `ke` pivot;
`ru-fe` exclusive scope; `helm` / `helms` identity spectrum.

Notation marks: evidential frame `()`; aside frame `[]`; approximation `~` / `ven`;
quotation `""`; prosodic suspension `—` / `el`; scope prefixes `a- i- u- o- e-`; topic
frame `:`.
