# Stage 2 — Compound construction

This is where structural thinking either clicks or breaks. If someone stops learning
Tonesu, it's usually here.

Topics: the head-final rule in depth; the `'` juncture marker; the `no-X` negation
pattern; the `[X]-no-fe` extremal pattern; derivational suffixes (`-li`, `-mu`, `-pa`,
`-ge`, `-ki`). And throughout: ambiguity resolution drills.

**Ambiguity drills** are the core exercise of this stage. Given a compound like
`to-fe-su-ki`, you identify 2–3 possible parses, determine which is correct and why,
and explain what each alternative would mean. This trains the structural reasoning
that Tonesu fluency requires.
