# Tonesu

The language itself — formalized. This section covers everything that is fixed and normative: the sounds, the grammar rules, the registered words, the notation.

If you are new here, start with [Sounds](sounds.md) and [Grammar](grammar.md).

---

## In this section

- **[Sounds](sounds.md)** — the phonology: consonants, vowels, syllable shapes, and the tier system
- **[Grammar](grammar.md)** — sentence structure, role particles, word order, and discourse markers
- **[Building words](words.md)** — how compounds are formed and parsed from primitive roots
- **[Notation](notation.md)** — the written marks: `~`, `'`, `()`, `::`, `;`, and others
- **[Primitives](primitives.md)** — the 34 foundational roots: the conceptual atoms of the language
- **[Registry](registry/overview.md)** — registered derived forms: the growing vocabulary built from primitives
- **[Corpus](../totonesu/corpus/overview.md)** — all attested sentences and conversations, organized by theme and batch
