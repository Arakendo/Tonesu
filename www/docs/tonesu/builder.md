# Word builder

Click CV roots from the palette to compose a compound word.
The Registry lookup panel shows whether the combination already has an entry.

All 34 active primitive roots are shown, colour-coded by consonant family.
Use the scope-prefix and suffix buttons to add grammatical modifiers.
