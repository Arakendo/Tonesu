## Composition

ne (relation/connection) + to (conceptual pattern) — relational pattern

## Definition

Predicate asserting that two or more entities stand in a pattern of mutual correspondence — they are similar or analogous to each other. Weaker than `to-ru` (W083, identity): `ne-to` claims a relational pattern, not a shared template. Used in plural-subject predication: `lo-pu-X  ne-to` = "the Xs are similar to each other."

## Notes

The head is `to` (the pattern); `ne` specifies the pattern is relational.
Contrast: `to-ru` = one unified pattern (identity); `ne-to` = relational pattern between distinct entities (similarity).

## Related
