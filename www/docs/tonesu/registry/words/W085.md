## Composition

su (structure/organization) + ru (unity/singularity) — unified structure

## Definition

A structured whole in which components are bound into a single organized system. Used as the referent in component-membership predicates: `la-X  pe lo-su-ru` = X is a component of one unified system. Distinct from `su-mu` (structural artifact = machine part) in that `su-ru` names the unity of the system itself, not a physical component.

## Notes

`pe` as stative predicate (`lo-X  pe  lo-su-ru`) = first corpus use of `pe` in the predicate slot (established as first-attested construction).
