## Composition

de (decrease/harm) + li (social agent / person). Head-final: li (agent) is head; de specifies the type of agent by effect — one whose agency causes decrease. Mirror of be-li (W056: benefactor = increase-agent).

## Definition

an agent whose defining effect on a counterpart is decrease or harm; adversary; opposing agent; enemy. Head li (social agent/person), modified by de (decrease/harm): the person whose agency causes decrease. Covers military enemy, legal adversary, competitive opponent — any agent positioned in opposition whose agency tends toward the counterpart's decrease. Distinct from de-zo-li (W089: personified death) which carries mythological loading; de-li is a relational role term.


## Notes

First attestation AOW-001-C (S704). Part of the know-yourself triptych (S705–S707). Used as la-de-li (the adversary, agent-marked) throughout S704–S707. Intentionally relational: de-li is "adversary" only relative to a counterpart — there is no absolute de-li in Tonesu, only contextually adversarial roles.

## Related

de (decrease/harm), li (social agent), be-li (W056: benefactor), de-zo-li (W089: personified death), ka-ra (W222: combat), de-ra (compositional: defeat / force-decrease)
