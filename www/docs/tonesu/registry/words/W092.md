## Composition

ka (intentional action) + ne (relation) + de (decay)

## Definition

A named agent deliberately caused a relational bond to dissolve. Used where English says "X broke up with Y" or "X ended things with Y." The `ka` root (intentional action, deliberate act) marks that the dissolution was chosen, not that it simply happened. Agent is marked `la-X`; patient relational territory is `lo-ne-Y` (the relation belonging to the patient).
`ka-ne-de` differs from bare `ne-de` (W091) exactly as "X intentionally broke the connection" differs from "the connection broke."

## Notes

Compositional parse: `ka` (intentional-action modifier) + `ne-de` (relation-decay compound). The `ka-` prefix lifts natural-process `ne-de` into the agentive register. Standard agentive frame: `la-[agent]  ka-ne-de lo-ne-[patient]`.
