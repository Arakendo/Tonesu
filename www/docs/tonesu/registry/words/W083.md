## Composition

to (conceptual pattern) + ru (unity/singularity) — unified pattern

## Definition

Predicate asserting that multiple entities share a single unified conceptual template: same design, same category, same type. Stronger than similarity (`ne-to`): `to-ru` claims identity at the pattern level, not merely analogy.
Used in plural-subject predication: `lo-pu-X  to-ru` = "the Xs are all the same model."

## Notes

Contrast with `ne-to` (W084, similarity — relational pattern, weaker).
`to-ru` is a unification claim; `ne-to` is a correspondence claim.

## Related

to (conceptual pattern), ru (unity), ne-to (W084), ru-to (unified concept = definition, W040)
