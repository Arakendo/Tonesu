# Translation Test: String Theory and the 11 Dimensions

## Source: Standard physics exposition — string theory and M-theory
## Status: Founding set (STR-001)

---

## Purpose

String theory is the theoretical physics stress test: a domain where the referents
are sub-observable (Planck-scale strings), the geometric claims are counter-intuitive
(11 dimensions when we observe 4), and the epistemological status is explicitly
uncertain (theorized, not empirically confirmed). This tests whether Tonesu can:

- **Express a new class of fundamental entity** — `ki-fe-mu` (the string) as a 1D
  oscillating boundary-artifact; distinct from macroscopic strings and from point
  particles
- **Handle dimension-counting** — positional digit notation (`bol bol` = 11) combined
  with the new compound `pa-di` (spatial direction) gives `bol bol nu pa-di` = eleven
  dimensions
- **Partition a count** — `mol` (4) observable + `yom` (7) compactified = `bol bol`
  (11) total; the arithmetic is visible in the sentence forms themselves
- **Express compactification** — `ko-pa-di` = `ko` (containment/enclosure) + `pa-di`
  (spatial direction) = an enclosed/rolled-up spatial direction; `ko` encodes the
  physical meaning exactly
- **Mark theoretical vs. confirmed** — evidential frame `()` in S731 marks string
  theory's unification claim as a held-model, not an observed fact; distinct from
  S730 which asserts force-emergence without hedge
- **Parallel `go {premise}, result` constructions** — S726 and S730 share the same
  causal framing (`go {ki-fe-mu}`) but assert different result-clauses (particles vs.
  forces), making explicit that strings are the unified ontological source of both

Primary tests:

1. Does `ki-fe-mu` cleanly name a 1D oscillating entity without needing a new primitive?
2. Does `pa-di` generalize correctly as "dimension" across mathematical, physical, and
   everyday uses?
3. Does the digit-arithmetic partition (`mol` + `yom` = `bol bol`) read correctly?
4. Does `ko-pa-di` encode compactification without a specialized physics vocabulary?
5. Does the evidential frame `()` carry the theoretical/unconfirmed status of string
   theory without needing a dedicated epistemic particle?

Corpus sentences: S726–S731

## The `ki-fe-mu` Design

**Why not a point-particle?**

Standard particle physics uses point-particles (`pe-ma`): zero-dimensional, no
spatial extent. String theory replaces these with `ki-fe-mu`: a 1D oscillating
entity. The Tonesu composition captures this:

| Component | Root | Contribution |
|-----------|------|-------------|
| `ki` | motion/oscillation | the string vibrates — oscillation is definitional, not accidental |
| `fe` | boundary/limit/1D | the string has exactly one spatial dimension — it is a "boundary" in space |
| `mu` | artifact/entity | a theoretical object; something that can be modeled and manipulated |

The right-branching parse: `ki` modifies [`fe-mu`] = "the 1D-boundary artifact in
oscillatory motion." `fe-mu` = the minimal spatial form — a 1D entity. `ki-fe-mu` =
the 1D entity that is dynamically in motion = a vibrational string.

**Why `fe` for one-dimensionality?**

`fe` = boundary/limit. A point has zero extent (too small for `fe` = no boundary to
speak of). A line or 1D string IS a boundary: it is the limit between two spatial
regions, extended in exactly one direction. The string in string theory is literally
this: a Planck-scale boundary-object oscillating in a higher-dimensional space.
`fe-mu` = the boundary-artifact = the 1D entity. This composition is not arbitrary —
it reflects the actual geometric character of the string.

**What `ki-fe-mu` is NOT:**

- Not a macroscopic string, rope, or filament — those are `su-ma` (structured matter)
  or domain-specific forms
- Not a particle (`pe-ma`) — the 1D vs. 0D distinction is structural
- Not a wave — `ki-be` (vibration mode) names the modal state; `ki-fe-mu` names the
  entity that vibrates

## The `pa-di` Design

**Dimension as spatial direction:**

In standard mathematics and physics, a "dimension" is an independent degree of
freedom. In 3D space: left-right, forward-back, up-down — three independent
directions. Each is a `pa-di`:

```
pa (space) + di (direction) = an independent direction through space = dimension
```

The composition is minimal and exact. No additional roots are needed. The count
construction reads naturally:

```
bol bol nu pa-di     → eleven spatial-directions → 11 dimensions
mol   nu pa-di       → four spatial-directions   → 4 dimensions
yom   nu ko-pa-di    → seven enclosed directions → 7 compactified dimensions
```

**The temporal dimension:**

Strictly, 4D spacetime has 3 spatial (`pa-di`) + 1 temporal (`ti-di`) dimensions.
`pa-di` is used inclusively in STR-001 for all 11 dimensions, following the physics
convention that "n dimensions" includes time. When precision is required: the
observable 4 consist of `gal nu pa-di` (three spatial) + `bol nu ti-di` (one
temporal). The STR-001 notes mark this distinction where relevant.

**Compactification via `ko`:**

```
ko-pa-di = ko (containment/enclosure) + pa-di (dimension)
         = an enclosed/contained spatial direction
         = a compactified dimension
```

The `ko` primitive encodes the physics with precision: compactification means a
spatial direction is "enclosed" — it curls back on itself at Planck scale, becoming
too small to access experimentally. `ko` = containment/interior is exactly this.
`ko-pa-di` vs `pa-di` is the compactified-vs-large dimension distinction.

## The Digit Arithmetic of 11 Dimensions

M-theory's 11 dimensions partition into:

| Count | Form | Dimensions |
|-------|------|-----------|
| 4 | `mol nu pa-di` | observable (3 spatial + 1 temporal) |
| 7 | `yom nu ko-pa-di` | compactified (too small to observe) |
| 11 | `bol bol nu pa-di` | total |

The arithmetic is visible in the digit forms: `mol` (4) + `yom` (7) = `bol bol` (11).
The partition in S728 and S729 is not just descriptive — it encodes the total in a
recoverable way. `bol bol` = positional digit notation for 11 (one-one in the
Tonesu digit system: most-significant `bol` first, second `bol` for the units).

## Structural Findings

### 1. `go {ki-fe-mu}` as unified causal frame (S726, S730)

S726 and S730 share the same causal premise but different result-clauses:

| Sentence | Form | Result | Reading |
|---------|------|--------|---------|
| S726 | `go {ki-fe-mu}, la-pe-ma ne ki-be` | particles are vibration-modes | particles emerge from strings |
| S730 | `go {ki-fe-mu}, la-o-ra be` | all forces arise | forces emerge from strings |

This parallelism encodes the unification thesis structurally: a single causal ground
(`ki-fe-mu`) is the premise for both the particle claim and the force claim. String
theory's central claim IS this parallelism — that one entity generates both sectors.

### 2. `be` as emergence predicate (S730)

`la-o-ra be` = "all forces grow/emerge." `be` (growth/increase/emergence) as a
standalone predicate: used previously to mark biological growth (`be-su`) and
structural completion. In S730, `be` serves as an emergence predicate: forces are
not previously-existing entities that strings merely organize — they arise FROM
strings. `be` = ontological emergence, not just increase. This extends the semantic
range of `be` into physics emergence.

### 3. Evidential frame `()` for theoretical status (S731)

S730 (`go {ki-fe-mu}, la-o-ra be`) asserts force-emergence without qualification.
S731 (`(go {ki-fe-mu}, la-o-ra ne su-ne)`) makes the stronger unification claim
but wraps it in `()`:

| Sentence | Form | Status |
|---------|------|--------|
| S730 | `go {ki-fe-mu}, la-o-ra be` | asserted directly — forces emerge from strings |
| S731 | `(go {ki-fe-mu}, la-o-ra ne su-ne)` | held theoretically — unification is a model |

The `()` frame marks string theory's epistemological status precisely: it is a held
model (reported, theorized) not an empirically confirmed fact. This is not hedging
for stylistic reasons — it is factually accurate. String theory is not yet
experimentally confirmed. The evidential frame carries this semantic load without
needing a dedicated "unconfirmed" or "theoretical" adjective.

### 4. `o-` scope prefix on `ra` (S730, S731)

`o-ra` = collective-force = all forces. The `o-` (collective scope prefix) applied to
a primitive root `ra` gives "all force collectively" — the complete force sector
(strong, weak, electromagnetic, gravity). This is a natural extension of the `o-`
prefix, previously attested on `o-zo-li` (collective-persons) and similar forms. The
scope prefix works here with a non-living root, confirming that `o-` is not limited
to agents or organisms — it collects over any domain.

## Step-by-Step Translation

### S726 — STR-001-A — Particles are vibration modes of strings

```
go {ki-fe-mu},  la-pe-ma  ne  ki-be
```

**Written:** `go {kifemu}, lapema ne kibe`

**Parse:**
- `go {}` — causal/origin frame
- `ki-fe-mu` — W228: fundamental string (premise = on account of strings)
- `la-pe-ma` — agent/topic: particles (matter-components)
- `ne` — property copula
- `ki-be` — vibration-mode (motion-structure)

**Notes:** The foundational claim of string theory. `pe-ma` = particle = component of
matter (anchor-inventory compositional). `ki-be` = motion-structure = a structured
mode of oscillatory behavior. The sentence asserts that the string (`ki-fe-mu`) is the
causal/ontological ground (`go {}`) and that particles have the property of being
vibration-modes of it. Different `ki-be` of the same `ki-fe-mu` give different
particles — the electron, the quark, the neutrino are all the same string vibrating
differently.

### S727 — STR-001-B — Space has 11 dimensions

```
la-pa  ne  bol bol nu pa-di
```

**Written:** `lapa ne bol bol nu padi`

**Parse:**
- `la-pa` — agent/topic: space (spatial primitive)
- `ne` — property copula
- `bol bol nu pa-di` — eleven measures of spatial-direction (W229)

**Notes:** W229 `pa-di` first use. `bol bol` = 11 in positional digit notation (digit
1 followed by digit 1 = one-one = eleven). The standard count construction: `{digits}
nu {noun}` = N of noun. `bol bol nu pa-di` = eleven-count of spatial-directions =
eleven dimensions. M-theory (the 11-dimensional version of string theory) is the
specific framework. The 11 dimensions include both 10 spatial + 1 temporal in strict
notation; `pa-di` used inclusively here following physics convention. See §Structural
findings for the `pa-di` vs `ti-di` note.

---

### S728 — STR-001-C — Four dimensions are observable

```
mol nu pa-di  ne  lu
```

**Written:** `mol nu padi ne lu`

**Parse:**
- `mol nu pa-di` — four spatial-directions (four dimensions)
- `ne` — property copula
- `lu` — visibility/observability (light primitive in property position)

**Notes:** `mol` = 4 (digit). `ne lu` = has the property of visibility = is observable.
The four large dimensions of spacetime (3 spatial + 1 temporal) are the ones we
inhabit, measure, and observe. `lu` in property position extends its root meaning
(light/visibility) to "observability" broadly — accessible to perception and
measurement, not just visible to the eye. This is the established semantic extension
of `lu` (from anchor-inventory: `nu lu` = quantity of light; property position = the
observable/perceptible condition). No `la-` agent prefix on the count-NP subject;
the NP-before-`ne` topic-comment construction is used.

### S729 — STR-001-D — Seven dimensions are compactified

```
yom nu ko-pa-di  ne  no-lu
```

**Written:** `yom nu kopadi ne nolu`

**Parse:**
- `yom nu ko-pa-di` — seven compactified-spatial-directions
- `ne` — property copula
- `no-lu` — non-observability (negation of visibility)

**Notes:** `yom` = 7 (digit). `ko-pa-di` = compactified dimension: `ko`
(containment/enclosure) + `pa-di` (W229) = an enclosed spatial direction = one that is
rolled up to Planck scale. `ne no-lu` = has the property of non-observability. The
`ko` primitive captures compactification structurally: these dimensions are not absent
— they are contained within Planck-scale geometry, inaccessible from the 4 large
dimensions. First use of `yom` (7) in corpus. S728/S729 pair forms the complete
partition: `mol` (4) + `yom` (7) = `bol bol` (11).

### S730 — STR-001-E — All forces arise from strings

```
go {ki-fe-mu},  la-o-ra  be
```

**Written:** `go {kifemu}, laora be`

**Parse:**
- `go {}` — causal/origin frame
- `ki-fe-mu` — W228: fundamental string (premise)
- `la-o-ra` — agent/topic: all forces (`o-` collective prefix + `ra` force)
- `be` — grow/emerge (predicate)

**Notes:** Parallel to S726: same `go {ki-fe-mu}` causal frame, different result-clause.
`o-ra` = collective-force = the full force sector: strong nuclear, weak nuclear,
electromagnetic, gravitational. `la-o-ra be` = all forces arise/emerge. `be` =
growth/emergence as predicate — forces are ontologically generated from strings, not
re-organized from prior entities. The S726+S730 pair is the structural encoding of
string theory's core claim: strings are the unified origin of BOTH particles and forces.

### S731 — STR-001-F — Theorized unification (evidential)

```
(go {ki-fe-mu},  la-o-ra  ne  su-ne)
```

**Written:** `(go {kifemu}, laora ne sune)`

**Parse:**
- `()` — evidential frame (reported/theorized, not directly asserted)
- `go {ki-fe-mu}` — causal frame: on account of strings
- `la-o-ra` — all forces
- `ne su-ne` — have structural-harmony (W174: unified)

**Notes:** The evidential frame `()` marks the content as a held theoretical model —
reported, proposed, theorized. String theory IS at this epistemic level: it is
mathematically rich and internally consistent but not yet confirmed by direct
experimental evidence (no string-scale experiments are possible with current
technology). The content inside `()` is `la-o-ra ne su-ne` = all forces are unified
in structural harmony. `su-ne` (W174) = structural harmony = the state of being in
harmonious structural relation = unified. `go {ki-fe-mu}` marks the mechanism: the
unification IS the string. So the full theoretical claim: "(by the mechanism of
strings, all forces are structurally unified)." This is what string theorists believe;
the `()` frame marks it as believed, not observed.
