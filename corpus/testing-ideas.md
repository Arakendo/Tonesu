## Ready to run — grammar structure gaps

- [ ] **Proof structure** — Multi-step deductive chains using `go`. Deferred pending subordinate clause resolution; that prerequisite is now done. Good target: Euclid's proof of infinite primes (short, requires nested premises, conclusion has the right kind of necessity). Test modus ponens, reductio ad absurdum, Cogito-style self-certifying inference. Contrast with `;` for mere sequence.
- [ ] **Permission vs capability modality** — No batch distinguishes "can" (physical capability, `be-vo`) from "may" (permission/authority). Flagged as "untested" in open questions. Good source material: medical consent, legal authority, social permission. Also stress-tests `wi-fe` from a different angle.

## Ready to run — new tier in use

- [ ] **CVCC anchors in running prose** — `varn` (π), `vern` (c), `birm` (ħ), etc. exist in inventory but appear in essentially zero running corpus sentences. Batch using them in actual calculation prose: area of a circle, relativistic mass, Planck's constant. Opens the question of how to embed numeric expressions like `vern = 2.998 × 10⁸ fe-ri`.

## Needs a small design decision first, then run

- [ ] **Performative / expressive register (EM-001)** — Reactive cries and expressive exclamations: "Ugh!", "Oh no…", "At last!" Not propositional — no standard `la-{agent} verb lo-{patient}` frame. Options: (a) bare topic frames (`ya, [state]`), (b) bare fragments with `—`, (c) new particle. Decision can be taken during the batch. Good source: stage directions / dramatic text.
- [ ] **Social / chosen kinship vocabulary** — "My spouse", "my adoptive parent", "my crew". No vocabulary exists. Sketch candidates first (`wi-ne`? → partner by bond; `zo-ne`? → chosen/formed relation), then run 6-sentence batch that forces them into use.

## Translation targets not yet attempted

- [ ] **Euclid — mathematical proof** — Infinite primes, or Book I Prop 1 (construct equilateral triangle). Short, precise, exercises nested `go`, `ne`, `wi-fe` (geometrical constraints). No mathematical prose translation series exists yet. Pairs with proof structure batch above.
- [ ] **News-item paragraph** — Neutral journalism register. Exercises `()` evidential frame at normal depth (FAL-006 tested adversarial depth-3; this is the complement). "It is reported that…", "sources say…", attribution chains.
- [ ] **Legal contract clause** — A clause from an actual legal instrument ("the landlord shall not enter without 24 hours' notice"). Tests `wi-fe` in its natural habitat, obligation vs prohibition, how date/duration is expressed. LGL-001 domain batch exists; this is a translation-register companion.
- [ ] **D&D — The Nine Alignments** — Law/Chaos and Good/Evil axes. The translation problem: `su` = order, `no-su` = chaos, `vo` = benefiting others, `de` = decrease/harm — but "neutral" is genuinely hard (absence? balance? indifference?). 9 sentences (one per alignment cell) plus a framing sentence. Cross-batch: Jedi/Sith can be mapped onto the grid. High philosophical load; pairs with the moral vocabulary from THO batches.
- [ ] **D&D — The Thirteen Damage Types** — Physical harm taxonomy: fire, cold, lightning, thunder, acid, poison, necrotic, radiant, psychic, force, bludgeoning, piercing, slashing. Tests whether primitive vocabulary covers physical interaction modes without borrowing. Several partially compositional (`ra-lu-ki` = fire, attested EXO-001).
- [ ] **D&D — The Fifteen Conditions** — Frightened, Charmed, Paralyzed, Blinded, Poisoned, etc. State vocabulary: transient properties imposed on a being. Stress-tests the `fa-` affect family and tests whether conditions distinguish from traits.
- [ ] **D&D — The Classes** — Fighter, Wizard, Rogue, Cleric, Druid, Paladin, Bard, etc. as social-role compounds via `-li` agent suffix. Simpler than other D&D batches; generates a clean role-registry.
- [ ] **D&D — The Great Wheel cosmology** — Planes (Material, Astral, Ethereal, Inner, Outer) as metaphysical-space vocabulary. High philosophical load; connects to scope-prefix tier (`a-`, `u-`, `o-`). Should wait until scope-discoverability (DISC-001) is resolved.
